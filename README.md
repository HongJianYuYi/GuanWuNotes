# 观物心易 · 学习笔记

> 基于梅花易数的起卦工具与学习笔记应用

---

## 简介

观物心易是一款基于梅花易数理论开发的起卦工具与学习笔记应用，支持多种起卦方法，提供电子书阅读和笔记管理功能，帮助用户深入学习和实践易学知识。

---

## 功能特性

### 起卦功能
- **时间起卦法**：根据当前时间自动计算卦象
- **三数起卦法**：通过输入三个数字进行起卦
- **端法起卦**：通过手动指定上下卦进行起卦

### 卦象展示
- 展示本卦、互卦、变卦、错卦、综卦
- 显示卦名、卦位、策数、轨数
- 自动计算农历日期

### 笔记管理
- 支持创建、编辑、删除笔记
- 笔记自动保存到 `notes` 文件夹
- 支持搜索笔记功能

### 电子书阅读
- 支持 PDF 格式电子书
- 左侧书架展示所有书籍
- 目录导航与页面跳转
- 书籍导入功能

---

## 技术栈

- **语言**：Python 3.8+
- **框架**：PyQt5
- **PDF解析**：PyMuPDF (fitz)
- **农历计算**：lunar-python

## 使用说明

### 起卦

1. 点击菜单栏「开始 → 起卦 / 刷新」或按快捷键
2. 选择起卦方法：时间起卦、三数起卦、端法起卦
3. 填写求测内容（可选）
4. 点击「开始起卦」按钮

### 笔记

1. 点击菜单栏「资料 → 笔记」
2. 在左侧列表选择或创建笔记
3. 在右侧编辑区域输入内容
4. 点击「保存笔记」按钮保存

### 电子书

1. 点击菜单栏「资料 → 电子书」
2. 点击「导入书籍」按钮选择 PDF 文件
3. 在左侧书架双击书籍打开阅读
4. 使用工具栏进行页面导航
